# 🔔 HealthCheck - Server Health Monitoring

**Preis: €9,99**

Ein umfassendes Health-Monitoring Tool für Server.

## Features
- CPU/RAM/Disk Check
- Service Status Monitoring
- Alert bei Problemen
- Telegram-Benachrichtigungen
- Web-Dashboard

## Installation
```bash
pip install -r requirements.txt
python3 healthcheck.py
```

## Konfiguration
Bearbeite `config.yaml` mit deinen Einstellungen.
