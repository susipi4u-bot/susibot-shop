# 📦 DockerWatch - Container Monitoring

**Preis: €8,99**

Monitor deine Docker-Container in Echtzeit.

## Features
- Container-Status
- Ressourcen-Nutzung
- Logs-Streaming
- Auto-Restart bei Crash
- Telegram-Alerts

## Installation
```bash
pip install -r requirements.txt
python3 dockerwatch.py
```
